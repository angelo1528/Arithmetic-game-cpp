# Menu-Driven Arithmetic Game (C++)

## Description
A console-based C++ application that generates random numbers and challenges the user to guess their sum or difference. The game features a menu-driven interface, input validation, and a dynamic scoring system.

## Features
- Random number generation (100–150 range)
- Menu-driven user interaction
- Input validation for user name and menu selection
- Scoring system with rewards and penalties
- Option to display numbers at a cost
- Replay and exit functionality

## How to Run
1. Compile:
   g++ main.cpp -o game
2. Run:
   ./game

## Concepts Used
- Control flow (loops, conditionals)
- Functions and enums
- Randomization
- Input validation
- State management
